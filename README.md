# Alexander Bertermann - Academic Website

## Upload to GitHub

1. Create new repository: `A-Bertermann.github.io`
2. Upload all files
3. Settings → Pages → Source: "Deploy from a branch" → Branch: main → Save
4. Wait 2 minutes → Site live at https://a-bertermann.github.io

## Files

- `index.html` - Main website (edit this!)
- `profile.jpg` - Your photo (add this!)
- `pdf/` - Your papers and CV (add PDFs here!)

## To customize

Edit `index.html`:
- Line 65: Your name
- Line 66: Your title  
- Lines 82-90: Your bio
- Line 97: Contact info
- Lines 107+: Your papers
